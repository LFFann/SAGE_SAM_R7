# SAGE-SAM R7

Standalone implementation of **SAGE-SAM R7** for semi-supervised medical image segmentation with class-balanced foreground-safe pseudo supervision and SAM-as-structural-verifier training.

See [README_R7.md](README_R7.md) for the new V100 training commands, validation, testing, and key diagnostic checks. The R6 notes are kept in [README_R6.md](README_R6.md) for compatibility context.
